## Instalação

Instructions on how to install and run the project locally. For example:

1. Clone the repository:
```bash
git clone https://github.com/mhhomura/recat-app-example.git
```
2. Get into the folder:
```bash
cd app-example
```
3. Install the dependencies:
```bash
yarn 
```
4. Run the project
```bash
yarn start
```
